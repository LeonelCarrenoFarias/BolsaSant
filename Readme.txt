Aplicación web desarrollada con HTML5, plantila boststrap, y Jquery.

El consumo de la api se efectuó por Jquery mediante $AJAX (post)

Al comienzo la desarrolle en PhP pero no pude obtener los datos con el metodo file_get_content debido a que el servidor de bolsa de santiago no permite este metodo.

Para vizualizar los datos se debe ingresar el token de acceso.

Existen dos pestañas, una con una tabla con todos los valores y otra con el grafico con los datos descargados.
